# **Cahier des Charges \- Tagada Studio**

## **1\. Présentation de la Société**

Tagada Studio est une structure spécialisée exclusivement dans la photographie dédiée aux univers :

* **Scolaire :** Écoles maternelles et élémentaires.  
* **Sport :** Clubs sportifs locaux.  
* **Associations :** Structures culturelles et sportives.

## **3\. Architecture SEO (Silos)**

Le site sera structuré pour maximiser la visibilité locale (Neuilly-sur-Marne / Île-de-France) :

* **Accueil :** Hub global sur l'expertise scolaire, sport et associatif.  
* **Page Scolaire :** Focus sur la gestion automatisée et le service clé en main.  
* **Page Sport :** Focus sur les composites, portraits et financement club.  
* **Page Association :** Focus sur le reportage événementiel et les galas.

## **5\. Base de Connaissance (FAQ pour intégration)**

### **Pourquoi choisir Tagada Studio ?**

Nous automatisons la gestion administrative pour vous libérer de toute charge, avec une qualité d'image professionnelle.

### **Comment se déroule une séance ?**

Organisation fluide : planification, prise de vue rapide, mise en ligne sécurisée, livraison sans effort.

## **6\. Directives Techniques pour GitHub Pages**

    \- Stack : HTML5, CSS (Tailwind via CDN pour la v1), JavaScript (léger)  
    \- Performance : Lazy loading pour les images, format WebP obligatoire.
